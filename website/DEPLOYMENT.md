## Deployment Options

### Option 1: Vercel (Recommended)
```bash
npm i -g vercel
cd cyberpunk-dist
vercel --prod
```

### Option 2: GitHub + Vercel
1. Push to GitHub
2. Connect to Vercel
3. Automatic deployment

### Option 3: Manual Upload
1. Zip the cyberpunk-dist directory
2. Upload to any static hosting
3. Ensure HTTPS for full features
